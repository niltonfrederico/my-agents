## Usage Patterns

### Quick Reference Lookups

- YAML syntax errors → `troubleshooting.md`
- applyTo pattern examples → `agent-patterns.md`  
- Tool restrictions → `tools-and-capabilities.md`
- Frontmatter properties → `customization-syntax.md`

### When Creating New Agents

1. Check `agent-patterns.md` for similar examples
2. Reference `customization-syntax.md` for proper YAML
3. Validate tool usage with `tools-and-capabilities.md`
4. Document new patterns in appropriate files
